# 📘 Introduction

**FanCode API** is a free IPTV & JSON service where you can:
- Get live match HLS links for IPTV players like **VLC, Tivimate, MX Player** etc.
- Use our **developer-friendly JSON API** to fetch match data for your app or website.
- Embed, stream, or analyze live matches using simple HTTP endpoints.

> Built for developers, sports lovers, and IPTV fans!

---

# 🔥 Features

- 🎥 IPTV M3U Link support (auto-generated per match)
- 📦 JSON API with match details, thumbnails & stream URLs
- 📲 Works on Android, TV, iOS, and PC
- 🆓 100% Free and open source
- ♻️ Regularly updated links

- **TV**: TiviMate, OTT Navigator
